## How to Run Locally

To view the website locally:

```bash
git clone https://github.com/yczhangcc/yuchengzhang_portfolio.git
cd yuchengzhang_portfolio
```

Then open `index.html` in a browser.

Since this is a static website, no backend server or package installation is required. A rare mercy in software development.

---

## Deployment

This website is deployed with **Vercel**.

After changes are pushed to the main branch, Vercel automatically rebuilds and updates the live website.
